compiling is made using Makefile,type make and the code gets compiled,

you can play the game by executing "ans" file using the command ./ans

you can clear the executable file using "make clean"

additional things are :
 you can pause a game in middle and start again.
 
 the game can restart when the game is over.

 the score is displayed in the game.

 mirrors keep rotating dynamically at a speed based on level.
